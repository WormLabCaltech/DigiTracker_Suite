คำอธิบายในการปล่อยตัวผลิตภัณฑ์นี้
========================

ซอฟท์แวร์ Matrox PowerDesk รวมทั้งไดรเวอร์จอภาพและยูทิลิตี้จอภาพ
ด้วยซอฟท์แวร์นี้
คุณสามารถได้รับประโยชน์จากฮาร์ดแวร์ Matrox graphics
และคุณสามารถเข้าไปเพิ่มฟีเจอร์ต่าง
ๆ ของ Matrox ที่สัมพันธ์กับจอภาพ


การติดตั้ง
=======

การติดตั้ง Matrox PowerDesk  เริ่มที่โปรแกรม "setup"  จากนั้นตามด้วยคำแนะต่าง
ๆ นำบนจอ

โปรแกรมเซตอัพจะติดตั้งซอฟท์แวร์เท่านั้น  ถ้า แบบ Matrox graphics card
ซัพพอร์ตโดยโปรแกรมเซตอัพที่ถูกติดตั้งในเครื่องคอมพิวเตอร์ของคุณ


[English...]

New features
============

This software includes new features, especially for Matrox 
hardware with DualHead support. These include:

- The use of different DualHead features (Multi-Display, DVDMax, 
  Zoom, and Clone) *without* restarting Windows

- Easier-to-use interface for DualHead controls

- Controls for dividing graphics resources (memory and bandwidth) 
  between your main and secondary displays

- More features accessible through user-defined hot keys 
  (keyboard shortcuts)

[...English]


ข้อมูลเพิ่มเติม
=========

สำหรับข้อมูลเพิ่มเติมเกี่ยวกับ Matrox PowerDesk โปรดดูที่ไฟล์ help
สำหรับข้อมูลเฉพาะMatrox
PowerDesk โปรดดูที่ Matrox หรือคู่มือระบบ


หมายเหตุปัญหาและขอบเขตของความสามารถ
============================

- การติดตั้งใน Windows รุ่นที่แปลออกเป็นภาษาอื่น

  ถ้าท่านติดตั้งซอฟท์แวร์ในภาษาที่ต่างจากภาษาของ
  ระบบปฏิบัติการของท่าน (ตัวอย่าง เช่นซอฟท์แวร์ภาษาอังกฤษ
  ติดตั้งในระบบที่ใช้ภาษาญี่ปุ่น)
  ท่านอาจมีปัญหากับข้อความ และกล่องไดอะลอกสำหรับควบคุมต่างๆ
  ถูกตัดทิ้งไปทั้งนี้เนื่องจากความแตกต่างของตัวอักษรสำหรับระบบ

- สนับสนุนไดรวเวอร์ OpenGL (3D)

  หากคุณมีผลิตภัณฑ์ Matrox รุ่นที่ใหม่กว่าเดิมหรือคุณได้รับซอฟท์แวร์
  Matrox มาจากผู้ผลิตเครื่องคอมพิวเตอร์ การสนับสนุนอย่างเต็มรูปแบบ
  ของ OpenGL อาจจะไม่ทำงานบนไดรเวอร์จอแสดงภาพ Matrox
  สำหรับรายละเอียดไดรเวอร์จอแสดงภาพ Matrox พร้อมการสนับสนุน
  แบบเต็มรูปแบบของ OpenGL ให้ดูที่เว็บไซต์ของ Matrox (www.
  matrox.com/mga)  (หากคุณได้รับผลิตภัณฑ์ Matrox จากผู้ผลิต
  เครื่องคอมพิวเตอร์ กรุณาตรวจสอบรายละเอียดเพิ่มเติมจากเว็บไซต์ของ
  ผู้ผลิตไดรเวอร์จอแสดงภาพด้วย ทั้งนี้ ไดรเวอร์จอแสดงภาพที่คุณ
  ได้รับจากผู้ผลิตเครื่องคอมพิวเตอร์ มักต้องผ่านการทดสอบร่วม
  กับรุ่นของคอมพิวเตอร์ของคุณก่อน)
